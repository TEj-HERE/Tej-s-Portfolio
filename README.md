# Tej-s-Portfolio
My Portfolio

## Local preview

- **Fastest**: open `index.html` directly in your browser.
- **Recommended (local server)**: use any static server (better for consistent loading).
  - If you have Python:
    - `python -m http.server 5500`
    - Then open `http://localhost:5500`

## Deploy (GitHub Pages)

- Push to GitHub, then enable Pages for the repo (root).
- The site is plain static files: `index.html`, `styles.css`, `script.js`.
